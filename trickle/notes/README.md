# Ryan Malatamban Portfolio Website

A professional portfolio website showcasing Ryan Malatamban's skills, experience, and certifications as a BSIT student and welder.

## Features

- Responsive design with mobile navigation
- Hero section with profile image and social media links
- About Me section with personal introduction
- Experience showcase (SMAW, JDVP, Work Immersion)
- Projects gallery with image viewer
- Certifications display with modal image viewing
- Contact form with mailto functionality
- Modern UI with Tailwind CSS

## Sections

1. **Header** - Fixed navigation with smooth scrolling
2. **Hero** - Introduction with profile photo and social links
3. **About** - Personal background and skills summary
4. **Experience** - Work experience and training details
5. **Projects** - Showcase of completed welding projects
6. **Certifications** - Display of earned certificates
7. **Contact** - Contact form for inquiries
8. **Footer** - Copyright information

## Technologies

- React 18
- Tailwind CSS
- Lucide Icons
- Responsive Design
